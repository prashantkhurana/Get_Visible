

The application is built upon digital self. The the introdution slideshow is run by http://lvh.me:8000/indexIntro.

First run MongoDb and then go to the application and run python manage.py runserver.

Now into your browser :  http://lvh.me:8000/indexIntro

From here onwards the application is intuitive.

The queries along with the hashmap implemnataion and grouping by hours are in the stats.py file. In views.py various arrays are created and sent to the user interface.
